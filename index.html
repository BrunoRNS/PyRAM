PYRAM - Lightweight PyPy Wrapper

PYRAM is an optimized version of PyPy, designed for server environments. It eliminates unnecessary components such as pip, setuptools, and tcl/tk, but includes essential packages like Django and PyMySQL pre-installed. By running the PyPy JIT directly in RAM, PYRAM ensures high-speed execution.

Features

  • Optimized PyPy wrapper for servers
  • Includes Django and PyMySQL pre-installed
  • Eliminates pip, setuptools, and tcl/tk
  • High-speed execution by running JIT directly in RAM

Installation

  1. Make sure you have sudo privileges.
  2. Download the package.
  3. Install the package: 
    sudo dpkg -i PyRAM.deb

Usage

After installation, you can run the pyram command:

sudo pyram /path/to/your/script.py

Why sudo?

To use the mount function: in sudo mount (the_ramdisk). More information in usr/share/doc/src

What if I need other library?

Download the .whl file, extract it, and download by your own the dependencies, and then use it normaly
